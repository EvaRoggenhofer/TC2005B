¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es una red que habilita la comunicación y acceso a recursos de datos. Se usan ciertos protocolos de comunicación. El world wide web es una gran cantidad de documentos interconectados (p.ej. por hyperlinks). En otras palabras: el Internet es el medio; el World Wide Web está en el internet.

¿Cuáles son las partes de un URL?

esquema (protocolo de comunicación; cifrado con "s" o no cifrado) (http(s)), prefix (www), dominio (w3schools.com: dirección de la computadora con la que estamos comunicando), port (port nr for http es 80), path (directorio en el servidor), filename (nombre del archivo);

Un URL puede tener variables: ...html?variable=valor... (p.j. en una búsqueda de Google)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET se utiliza para solicitar datos de un recurso especificado del sevidor. Se usa cuando buscamos algo en el navegador. GET: solo tiene head, no body POST se utiliza para enviar datos a un servidor para crear/actualizar un recurso. POST si tiene body (= cifrado en https). formas: con POST. HEAD es casi idéntico a GET, pero sin el cuerpo de la respuesta. Si GET /users devuelve una lista de usuarios, entonces HEAD /users hará la misma petición pero no devolverá la lista de usuarios. Las peticiones HEAD son útiles para comprobar lo que devolverá una petición GET antes de hacer realmente una petición GET - como antes de descargar un archivo grande o un cuerpo de respuesta. PUT se utiliza para enviar datos a un servidor para crear/actualizar un recurso. La diferencia entre POST y PUT es que las peticiones PUT son idempotentes. Es decir, llamar a la misma solicitud PUT varias veces siempre producirá el mismo resultado. Por el contrario, llamar a una petición POST repetidamente tiene efectos secundarios de crear el mismo recurso varias veces. El método PATCH se utiliza para aplicar modificaciones parciales a un recurso. El método DELETE elimina el recurso especificado.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?
- Unidades físicas como pt sólo deben utilizarse para las hojas de estilo de impresión. Los píxeles px sólo deben utilizarse para la pantalla. Las unidades relativas como % se adaptan bien tanto a los tipos de medios de pantalla como de impresión y deberían ser las unidades CSS más utilizadas.
fuente
¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?
- Minimizar un CSS: eliminación del código innecesario de los archivos fuente CSS para reducir su tamaño (sin cambiar el funcionamiento del archivo CSS en el navegador). Esto ayuda al navegador a descargar y procesar estos archivos más rápidamente, lo que aumenta el rendimiento de la página (y, por tanto, también la experiencia del usuario).
