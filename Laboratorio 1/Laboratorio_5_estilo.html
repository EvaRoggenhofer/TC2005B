Frameworks de estilo: Bulma!

Pregunta: Describe "Material Design"

Eva Roggenhofer @A01762592 QRO
Material Design es un lenguaje de diseño para Android. Fue creado por Google en 2014.
Cuenta con efectos 3D, iluminación realista y funciones de animación en GUI inmersivas.
El objetivo es ofrecer resultados de alta calidad en todas las plataformas.
Material Design tiene componentes que se comportan como objetos reales;
al imitar el mundo físico, el objetivo es reducir la carga cognitiva de los usuarios.

Esta página está diseñada con BULMA.
Bulma es un framework gratuito y de código abierto que proporciona componentes frontales listos para usar que puedes combinar fácilmente para crear interfaces web responsivas.
El elemento box es un simple contenedor con un fondo blanco, algo de relleno y una sombra de caja.
